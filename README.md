# ReflectionSettings

> Contributions are more than welcome! 

### Importing ReflectionSettings into your project:
```
repositories {
    maven { url = 'https://jitpack.io' }
}
dependencies { 
    implementation 'com.github.CafeDevelopment:ReflectionSettings:VERSION'
}
```
